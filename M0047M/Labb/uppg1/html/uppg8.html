Hitta kritiska punkter för funktionen 

x = linspace(-5, 5, 2000);
f = @(x) x - sin(x ./ (x.^2 + x + 1));

figure;
plot(x, f(x));
xlabel('x');
ylabel('y');
grid on;
axis([-1 0.6, -1 1]);

Här kan man se att det finns kritiska punkter runt och .

För 

figure;
plot(x, f(x));
hold on;
x1 = 0;
xlabel('x');
ylabel('y');
grid on;
plot(x1, f(x1), 'o', 'MarkerSize', 3);
axis([-0.1 0.1, -0.1 0.1]);
hold off;

För 

figure;
plot(x, f(x));
hold on;
x2 = -0.518;
xlabel('x');
ylabel('y');
grid on;
plot(x2, f(x2), 'o', 'MarkerSize', 3);
axis([-0.6 -0.4, 0 0.20]);
hold off;
